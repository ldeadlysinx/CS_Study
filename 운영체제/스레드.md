# 스레드
정의) 프로세스를 구성하는 실행 흐름의 단위
하나의 프로세스는 하나 이상의 스레드를 가질 수 있다.

단일 스레드 프로세스
-실행 흐름이 하나인 프로세스

멀티 스레드 프로세스
-실행 흐름이 여러개인 프로세스(프로세스를 이루는 여러 명령어 동시 실행 가능)

구성요소(실행할 필요한 최소한의 단위)
- 스레드 ID
- 프로그램 카운터를 비롯한 레지스터 값
- 스택

프로세스를 이루는 스레드는 코드영역 데이터영역 힙 영역등을 공유한다.
별개의 레지스터 스택 프로그램 카운터를 갖는다.

장점)
 - 동시에 실행되므로 시간단축이 가능하다.
 - 메모리 낭비를 방지

단점)
- 데이터를 공유하므로 프로세스가 문제가 생기면 스레드가 문제가 생길수 있다.

프로세스들끼리는 자원을 공유하지 않는다 스레드끼리는 자원을 공유한다
cf)프로세스들끼리 자원을 공유는 icp 통신 이용해서 공유 가능하긴 하다. 잘 사용 x
